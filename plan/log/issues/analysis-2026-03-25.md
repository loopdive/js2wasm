## New Issues Created

1. **#789** -- TypeError null/undefined guard over-triggering (15,630 fail, CRITICAL)
   - The null guard system from #775/#780/#781 uses `ref.test $Struct` which fails for valid objects of different struct types
   - Fix: use `ref.is_null` instead of `ref.test` for the null guard

2. **#790** -- assert.throws(ReferenceError) not implemented (788 fail, HIGH)

   - Consolidates #429, #443, #723

   - Residual after #784 (done)
