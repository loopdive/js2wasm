### Diagnostic tool: local IR-vs-legacy SHA comparison

When CI shows regressions on a PR you don't think can cause them, locally compile each regressed test with both `experimentalIR: true` and `experimentalIR: false` and SHA the binaries. If they're identical to the PR's wasm output, the "regressions" are baseline drift (not real). PR #142 used this technique successfully — PRs #145 and #151 likely benefit from it too.

### bash cwd reset trap

`/workspace/.tmp/` and `/workspace/.claude/worktrees/<branch>/.tmp/` are different directories. The persistent shell cwd resets to `/workspace` between Bash tool calls, so `cat > .tmp/foo.mts` from a fresh prompt creates `/workspace/.tmp/foo.mts`, not the worktree's. Always prefix with `cd /workspace/.claude/worktrees/<branch> && ...` or use absolute paths.
