# Issue 93: Test262 coverage — built-ins/Object

## Summary

Add `built-ins/Object/*` to the test262 runner. We already support `Object.keys`, `Object.values`, `Object.entries` via host imports.

## Motivation

Object is the base of all JS values. Basic property access and Object static methods are used everywhere.

## Subcategories to add

- `built-ins/Object/keys`
- `built-ins/Object/values`
- `built-ins/Object/entries`
- `built-ins/Object/assign`
- `built-ins/Object/create`
- `built-ins/Object/freeze`, `built-ins/Object/isFrozen`

## Approach

1. Add paths to `TEST_CATEGORIES`
2. Run and triage
3. Most tests will likely need features we don't support — focus on what passes

## Complexity

S — mostly triage, few fixes expected

## Dependencies

None
