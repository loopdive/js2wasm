# #150 — Unsupported ClassDeclaration in expression positions

## Status: open

## Problem
41 test262 compile errors: "Unsupported statement: ClassDeclaration". Classes declared inside `for` loops, `if` blocks, or other statement positions aren't handled.

Example:
```javascript
for (var i = 0; i < 1; i++) {
  class C { }
  new C();
}
```

## Fix
Handle ClassDeclaration in all statement positions, not just top-level and function body.
