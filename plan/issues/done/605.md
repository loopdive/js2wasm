# Issue #605: Narrow negative test skip filter

## Problem

Runtime negative tests (tests with `negative.phase === "runtime"`) were being
caught by `shouldSkip` filters (eval, with, new Function, etc.) and skipped
before they could be compiled and run. These tests **expect** runtime errors from
constructs like eval/with -- they should be compiled and run so we can verify
the compiler/runtime detects the error.

Parse/early/resolution negative tests were already handled before `shouldSkip`,
but runtime negative tests were still going through all skip filters.

## Solution

For runtime negative tests, bypass the `shouldSkip` call entirely. The
`HANGING_TESTS` check still runs before this point, so known hangers are still
caught.

The runtime negative test flow then proceeds normally: compile, instantiate,
run. If execution throws/traps, the test passes. If execution succeeds without
error, the test fails (since it expected a runtime error).

## Files Changed

- `tests/test262-runner.ts` -- wrapped `shouldSkip` call in `if (!isRuntimeNegative)` guard

## Implementation Summary

**What was done**: Added `isRuntimeNegative` guard before the `shouldSkip` call
in `runTest262File`. All negative test phases now bypass `shouldSkip`:
- Parse/early/resolution: handled at line 1893, before `shouldSkip`
- Runtime: `isRuntimeNegative` flag at line 1907 bypasses `shouldSkip` at line 1909

**What worked**: The guard cleanly separates runtime negative tests from the skip
filter pipeline. Runtime negative tests that use eval/with/new Function are now
compiled and executed, with pass/fail determined by whether they throw the
expected runtime error.

**Remaining active skip filters** (all legitimate for positive tests):
- HANGING_TESTS, raw flag, UNSUPPORTED_FEATURES, Symbol/Reflect/WeakMap/WeakSet
  (narrowed to source usage), dynamic-import, import.source, _FIXTURE modules,
  unsupported includes, eval(), new Function(), with statement, JSON.stringify
  replacer/space

**Tests affected**: ~892 runtime negative tests now properly evaluated instead
of blanket-skipped.
