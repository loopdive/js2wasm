# Issue #562: Math special values + string concat null/undefined coercion

## Problem

Several test262 tests fail with "returned 0" (wrong results). Two fixable patterns identified:

1. **Math.atanh/expm1 special values**: Both functions failed to preserve negative zero (-0) and expm1 didn't handle +Infinity.
2. **String + null/undefined coercion**: `"1" + null` should produce `"1null"` and `"1" + undefined` should produce `"1undefined"`, but the compiler either folded them numerically (tryStaticToNumber treated string literals as numbers in `+` context) or passed null externref to the string concat import without coercing to "null"/"undefined" strings.

## Implementation Summary

### What was done

1. **Math.atanh**: Added check for `x == 0` returning `x` (preserves -0 sign via identity) before the main formula.

2. **Math.expm1**: Added check for `x == +Infinity` returning `+Infinity`, and `x == 0` returning `x` (preserves -0).

3. **tryStaticToNumber constant folding**: Added guard to prevent folding `string_literal + anything` as numeric. In JS, `"1" + undefined` is string concatenation producing `"1undefined"`, not `NaN`. The guard checks for string literals on either side of a `+` operator and returns `undefined` (no fold).

4. **String concat null/undefined coercion**: In `compileStringBinaryOp` (non-fast mode), added handling for externref operands with TS type `null` or `undefined`. These now drop the null externref and push the appropriate string constant (`"null"` or `"undefined"`) before calling the concat import.

### What worked
- All fixes are minimal and targeted
- No regressions in existing tests (math-inline, math-minmax, string-coercion, bitwise, anon-struct)

### Files changed
- `src/codegen/math-helpers.ts` — atanh and expm1 special value handling
- `src/codegen/expressions.ts` — tryStaticToNumber guard, string concat null/undefined coercion
- `tests/math-inline.test.ts` — added 12 new tests for atanh/expm1 special values
- `tests/string-coercion.test.ts` — added 4 new tests for string + null/undefined

### Tests now passing
- Math.atanh: NaN, +Infinity, -Infinity, |x|>1, -0, +0
- Math.expm1: NaN, -Infinity, +Infinity, -0, +0
- String concat: "1"+null, null+"1", "1"+undefined, undefined+"1"
