# #151 — `this` keyword in class methods for test262

## Status: open

## Problem
35 test262 compile errors: "'this' implicitly has type 'any' because it does not have a type annotation". In allowJs mode, `this` in class methods should be implicitly typed as the class instance.

## Fix
In allowJs mode, suppress the noImplicitThis error and infer `this` type from the enclosing class or function context.
