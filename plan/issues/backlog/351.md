# #351 -- Async iteration (for-await-of)

## Status: open

329 tests need for-await-of and async generators. Depends on both async support (#339) and iterator protocol (#345).

## Details

Async iteration combines two complex features:
1. The async function model (await, Promise-based execution)
2. The iterator protocol (Symbol.asyncIterator, next() returning Promises)

```javascript
async function* gen() {
  yield 1;
  yield 2;
}

for await (const val of gen()) {
  console.log(val);
}
```

This requires:
- Async generator functions (combine generator state machine with async state machine)
- `for-await-of` statement compilation
- Symbol.asyncIterator support
- Promise-based iterator protocol

This is one of the most complex features to implement and has two hard dependencies.
