# #354 -- Reflect.construct

## Status: open

120 tests use Reflect.construct(). Needs Reflect API support. Depends on prototype chain support (#343) because Reflect.construct allows specifying a newTarget for prototype chain manipulation.

## Details

`Reflect.construct(target, args, newTarget)` creates a new instance:
```javascript
function Foo(x) { this.x = x; }
var obj = Reflect.construct(Foo, [42]);
obj.x === 42;  // true
obj instanceof Foo;  // true
```

With `newTarget`:
```javascript
function Bar() {}
var obj = Reflect.construct(Foo, [42], Bar);
obj instanceof Bar;  // true (prototype comes from Bar)
```

Implementation requires:
1. Dynamic constructor invocation (calling a function as a constructor at runtime)
2. Prototype chain manipulation (setting the prototype based on newTarget)
3. Argument spreading from an array
