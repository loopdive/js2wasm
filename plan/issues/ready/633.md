# #633 — Object.defineProperty tests fail (297 FAIL)

## Status: open

297 tests involving Object.defineProperty fail at runtime. Property descriptors (writable, enumerable, configurable, get/set) are not fully applied to struct-backed objects.

### Root cause
Object.defineProperty is stubbed as a no-op or only handles simple cases. Tests expect descriptor effects (non-writable properties throw on assignment, non-enumerable properties not in for-in, etc.).
