# #627 — Wasm stack underflow: not enough arguments (354 CE)

## Status: open

354 tests fail with "not enough arguments on the stack for drop/local.set". Residual after #617 fixed the async void case. Remaining patterns involve:
- Conditional expressions where one branch is void
- Short-circuit evaluation (&& / ||) with mixed void/value branches
- Generator yield in expression position

### Fix
Track stack depth more carefully in conditional/short-circuit codepaths.
