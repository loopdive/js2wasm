# #405 — Internal compiler errors (64 CE)

## Status: open

64 tests fail with "Internal error compiling expression: Cannot read properties of undefined". The compiler crashes when encountering unexpected AST shapes that it does not handle.

## Details

These are defensive programming issues where the compiler accesses properties on AST nodes without checking for undefined first. Common patterns:

- Accessing `.name` on a node that may not have a name (computed properties, destructuring)
- Accessing `.type` on an expression whose TypeScript type resolves to undefined
- Accessing `.parent` or `.symbol` on nodes where the checker returns undefined
- Array indexing on parameters/arguments that may be empty

Each crash prevents compilation of the entire test file, so fixing these guards has outsized impact.
