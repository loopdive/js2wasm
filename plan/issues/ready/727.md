# #727 -- Sub-classify assertion failures (11,480 tests return wrong values)

## Status: open

## Problem

11,480 tests fail with "assertion failed" messages that include line information, meaning the test compiled and ran but returned the wrong value. This is the single largest failure bucket and needs sub-classification to identify which codegen features cause the most failures.

### What needs to happen

Run error analysis on the 11,480 assertion failures to sub-classify by what the test checks. Proposed categories:

| Category | Expected pattern | Likely root cause |
|----------|-----------------|-------------------|
| Property descriptor | `writable`, `configurable`, `enumerable` | Missing Object.getOwnPropertyDescriptor |
| Constructor/prototype | `instanceof`, `.constructor ===`, `.prototype` | Incomplete prototype chain |
| String method result | `.includes()`, `.indexOf()`, `.slice()` result wrong | String method edge cases |
| Numeric precision | `Math.`, `toFixed`, `toPrecision` wrong | Float edge cases |
| Array method result | `.map()`, `.filter()`, `.reduce()` wrong | Array method edge cases |
| Type coercion | `==`, `===`, `typeof` wrong | Coercion gaps |
| Iterator/generator | `next()`, `done`, `value` wrong | Generator state machine |
| Regex | `.test()`, `.exec()`, `.match()` wrong | Regex support gaps |
| Date | `new Date()`, `.getTime()` wrong | Date support gaps |
| Error type | `instanceof TypeError` etc. wrong | Exception payload missing type tag |
| Symbol | `Symbol.`, `@@iterator` wrong | Symbol support gaps |
| Proxy/Reflect | trap behavior wrong | Proxy not implemented |

### Approach

1. Extract the assertion line from each failure message
2. Map assertion lines to test source to identify what is being checked
3. Group by the checked feature/API
4. Count each sub-category
5. Create targeted issues for the top 5 sub-categories

### Deliverable

A report in `benchmarks/results/` with the sub-classification breakdown and recommended issues to create.

## Complexity: S (analysis only, no code changes)
