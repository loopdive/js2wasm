# #359 -- Object mutability methods (Object.freeze/seal/preventExtensions)

## Status: open

31 tests need Object.freeze, Object.seal, Object.preventExtensions, Object.isFrozen, Object.isSealed, and Object.isExtensible.

## Details

These methods control object mutability:
```javascript
const obj = { x: 1, y: 2 };
Object.freeze(obj);
obj.x = 10; // silently fails in sloppy mode, throws in strict mode
Object.isFrozen(obj); // true
```

Implementation approach:
1. Add a mutability flags field to struct-based objects (frozen, sealed, non-extensible)
2. Object.freeze: set frozen flag, make all properties read-only
3. Object.seal: set sealed flag, make all properties non-configurable
4. Object.preventExtensions: set non-extensible flag
5. Property set operations check these flags and throw TypeError in strict mode
