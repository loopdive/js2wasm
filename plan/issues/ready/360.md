# #360 -- JSON.stringify result comparison

## Status: open

30 tests compare JSON.stringify output. May need runner harness adjustment or full JSON.stringify support.

## Details

Tests use JSON.stringify to compare complex values:
```javascript
assert.sameValue(JSON.stringify([1, 2, 3]), '[1,2,3]');
assert.sameValue(JSON.stringify({a: 1}), '{"a":1}');
```

Implementation approaches:
1. **Full JSON.stringify** -- implement serialization for numbers, strings, booleans, null, arrays, and objects
2. **Runner harness** -- if tests only use JSON.stringify for comparison, could intercept in the test runner

A pure Wasm implementation would recursively walk the value, building a string using string concatenation primitives already available.

## Complexity: M
