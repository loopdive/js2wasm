# Object.defineProperty host import (#747)

## Problem

Object.defineProperty is used by ~1,931 test262 tests. For externref objects:
- Value descriptors used `__extern_set` (simple assignment, ignored flags)
- Flag-only descriptors (writable, enumerable, configurable) were no-ops
- `Object.getOwnPropertyDescriptor` returned null/undefined

## Solution

1. Add `__defineProperty_value(obj, prop, value, flags)` host import
   - Flags bitmask encodes writable/enumerable/configurable and whether each is specified
   - Actually calls `Object.defineProperty` on the JS side with proper descriptor
2. Add `__getOwnPropertyDescriptor(obj, prop)` host import
   - Returns the real property descriptor as externref
3. Update `Reflect.getOwnPropertyDescriptor` to rewrite to `Object.getOwnPropertyDescriptor`

## Files changed

- `src/codegen/object-ops.ts` — externref paths now call `__defineProperty_value`
- `src/codegen/expressions.ts` — `getOwnPropertyDescriptor` uses host import
- `src/runtime.ts` — new builtin handlers
- `tests/equivalence/helpers.ts` — add new imports to test env
