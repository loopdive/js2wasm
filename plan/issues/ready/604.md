# #604 — Narrow async flag skip filter (1,311 tests)

## Status: review

1,311 tests skipped because they have the `async` flag in metadata. The compiler now supports async/await (#30). Many of these should compile — the filter is a blanket skip from before async was implemented.

## Approach

1. Remove the `async` flag skip entirely
2. Tests that use simple async/await will pass
3. Tests with complex Promise chains, .then(), or async iteration may fail — file those as separate issues
4. Keep `unsupported feature: async-iteration` (94 tests) as a separate narrower filter if needed

## Complexity: S

## Implementation Summary

### What was done

Investigation revealed that the blanket async flag skip was **already removed** (comment at line 118 of test262-runner.ts confirms this). The remaining async-flagged tests being skipped were blocked for other reasons:

- 264 tests: `unsupported include: asyncHelpers.js` -- the test262 harness file was not in the allowed includes set
- 355 tests: `uses dynamic import() in source` (legitimate skip)
- 65+57 tests: `Promise.any` / `Promise.allSettled` (legitimate unsupported features)
- Other legitimate skips (eval, SharedArrayBuffer, top-level-await, etc.)

The actionable fix was to:
1. Add `asyncHelpers.js` to the allowed includes set in `shouldSkip`
2. Add an `asyncTest` shim in `wrapTest` that calls the test function synchronously (since async functions compile to regular functions, `.then()` chains from the real asyncTest would break)
3. Ensure `$DONE` is available when `asyncTest` is used (asyncTest calls `$DONE()` on success and `$DONE(err)` on failure)

This unblocks **264 tests** that were skipped solely because of the missing `asyncHelpers.js` include.

### What worked
- The `asyncTest` shim is simple: call `fn()` synchronously, catch errors, call `$DONE` appropriately
- The `$DONE` shim was already in place for direct `$DONE` usage

### What didn't apply
- The issue description's count of 1,311 included tests skipped for other legitimate reasons (dynamic import, Promise features, etc.) that cannot be unblocked by this change alone

### Files changed
- `tests/test262-runner.ts`: Added `asyncHelpers.js` to allowed includes, added `asyncTest` shim
