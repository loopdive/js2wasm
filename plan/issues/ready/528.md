# #528 — Test262 runner: show progress when starting each batch

## Status: open

The runner shows no output until the first category batch completes (~30-60s). Users see "Running 22972 tests..." then silence. It looks hung.

## Fix

Print a line when each batch starts:
```
  [ 0%] Math (273 tests)...
```

Also print when processing priority (previously-failed) tests:
```
  Retesting 30 previously-failed tests...
```

Currently the runner only prints after a batch finishes. Adding a start line gives immediate feedback.
