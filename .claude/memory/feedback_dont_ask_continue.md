name: feedback_dont_ask_continue
description: Don't ask "should I continue?" — keep dispatching work automatically
type: feedback
---

When working through a list of issues, don't pause to ask if we should continue. Keep dispatching new agents to freed slots automatically. Only stop if blocked or if there's a decision that requires user input.

**Why:** User wants continuous autonomous execution, not stop-and-ask at every milestone.

**How to apply:** After merging an agent's work, immediately launch the next issue. After test results come in, note them and keep going. Only ask when there's a real ambiguity or risk.
