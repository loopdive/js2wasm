# §29 Memory Model

**Spec**: https://tc39.es/ecma262/#sec-memory-model
**Status**: ➖ not applicable
**Test262 categories**: (none — covered transitively)
**Coverage**: not directly measured

## What the spec requires

The ECMAScript memory model is defined for shared memory + Atomics. Since we do not support SharedArrayBuffer, the memory model is trivially satisfied (single-thread sequenced).
